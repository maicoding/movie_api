A new movie api is created here.

Request URL "HTTP
Method" Request body data format Response body data format
Get a list of movies /movies GET None A JSON object holding data about all the movies
Get data about a single movie by title /movies/[Title] GET None "A JSON object holding data about a single movie, containing a title, director name, and genres property. Example:
{
""Title"": ""Insomnia"",
""Director"": {
""Name"": ""Christopher Nolan""
},
""Genre"": {
""Name"": ""Thriller""
}
}"
Get information on a movie genre /genres/[genre] GET None "A JSON object holding data about a genre, containing a name, and a description. Example:
{
""name"": ""Drama"",
""description"": ""Drama is a category or genre of narrative fiction (or semi-fiction) intended to be more serious than humorous in tone.""
}"
Get information on a single director by name /directors/[name] GET None "A JSON object holding data about a director, containing a name, a birth date and biographical information. Example:
{
""name"": ""David Fincher"",
""birthdate"": ""28.07.1962"",
""bio"": ""David Fincher is an American filmmaker and director. He is known for his work in the genre of crime film.""
}"
Add a user /users POST "A JSON object holding data about the user to add, structured like:
{
""name"": ""Niklas"",
""favouriteMovies"":[""Fight Club""]
}" "A JSON object holding data about the user who was added, including an ID:
{
""name"": ""Niklas"",
""favouriteMovies"": [
""Fight Club""
],
""id"": ""6f583c9e-ff2d-4460-9fb3-686f8b713395""
}"
Update a user name /users/[id] PUT "A JSON object holding data about the user name to change, structured like:
{
""name"":""Franz""
}" "A JSON object holding data about the user who was added, including an ID:
{
""id"": ""1"",
""name"": {
""name"": ""Franz""
},
""favouriteMovies"": [
""Pulp Fiction"",
""Insomnia""
]
}"
Update favourite movie list /users/[id]/favourite/[movieId] PUT None A text message indicating whether the movie was successfully added
Delete movie from favourite movie list /users/[id]/favourite/[movieId] DELETE None A text message indicating whether the movie was successfully deleted
Delete a user by ID /users/[id] None A text message confirming that the user was deleted
